# Stylish Portfolio Style Website Theme
Stylish is a template setup for new clients who want a basic information website from FAI Marketing Inc.

## Setup Stylish ##
_Here are the steps to install Stylish on Clients Server_
```
1) Setup Ubuntu Server
2) Install Apache2
3) Install PhP
4) Obtain SSL Certification
5) Alter Stylish Theme Wordage & Images for Client Needs
6) Push Updated Code to new Git Repo
7) Push Updated Code to Server
8) Finish off any extra Security
```
